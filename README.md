# GRYF Volleyball Camp — Debica 2026

Professional volleyball camp website built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Image Setup

Place the following images in `/public/assets/`:

| Filename           | Usage                       |
| ------------------ | --------------------------- |
| `logo_icon.png`    | Navbar logo, mobile menu    |
| `logo_main.png`    | About section, footer       |
| `game_1.jpg`       | Hero background             |
| `game_2.jpg`       | Features section            |
| `aleksandra.png`   | Coach profile card          |
| `gabba.png`        | Coach profile card          |

## Environment Variables

Copy `.env.local.example` to `.env.local` and set:

```
NEXT_PUBLIC_SIGNUP_FORM_URL=https://your-signup-form-url.com
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Mobile-first responsive design

## Build

```bash
npm run build
npm start
```
